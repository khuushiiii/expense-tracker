# Expense Tracker

A lightweight, fully client-side expense tracker built with HTML, CSS, and JavaScript — add, edit, and delete daily expenses with no backend required.

## Overview

A simple, responsive expense tracker that runs entirely in the browser. Every expense is added, edited, or deleted through direct DOM manipulation with no server or database dependency, making it fast to load and easy to deploy anywhere as a static site.

## Features

- Add, edit, and delete expenses
- Responsive design that works on both desktop and mobile
- Real-time UI updates on every change
- No backend or build step required — pure HTML, CSS, and JavaScript

## Getting Started

### Run locally

```bash
git clone https://github.com/khuushiiii/expense-tracker.git
cd expense-tracker
```

Then simply open `index.html` in your browser — no build step or dependencies needed.
